moov_cheerio

moov_cheerio is an extension of Cheerio, a lean server-side implementation of jQuery. moov_cheerio contains additional prototype methods, their documentation, and augmented documentation of the original Cheerio methods as well.

Documentation

All documentation examples will refer to the following HTML input in a standard webpage:

<ul id="fruits">
  <li class="apple" data-which="fuji">Apple</li>
  <li class="orange">Orange</li>
  <li class="pear">Pear</li>
</ul>

Development

Quick Start

After entering your project's directory, you can install your required dependencies locally using:

$ npm install

Run Tests

To run the test suite, enter the following command in your Terminal:

$ npm test
