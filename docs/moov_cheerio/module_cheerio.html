.data(nameopt, valueopt) Source (L35)

Gets or sets data bound to the first element in the matched set. Initially, the getters (empty or nonempty) will output an object containing attribute-value pairs from any data- attributes set in the HTML. Using the setter does not modify any HTML data- attributes, but instead binds the data to an internal structure for the Cheerio element object (using the data- attributes as a baseline). Using either getter (empty or nonempty) after using a setter outputs that updated internal data structure, including any data- attribute grabs that may have been overridden, although the data- attributes in the HTML will be retained.

Parameters
Name Type Attributes Description
name String <optional>
 A string for a data attribute name.
value String <optional>
 A string for the new data attribute's value.
Return Value

An object containing the attribute-value pairs for the data bound to the first element in the matched set (empty getter), a string containing the value for the given input attribute of the first element in the matched set (nonempty getter), or the updated form of the original Cheerio matched-set object (setter).

Examples
$body.find(".computer").data(); // getter for all attributes in an object
$body.find(".computer").data("apple-color"); // getter for a specific data- attribute
let apple = $body.find(".computer").data("apple-size", "2TB"); // setter
let apple = $body.find(".computer").data("apple-color", "green"); // setter

Adds or removes class(es) from each element in the matched set, depending on either the class's presence or the value of the switch argument.

Parameters
Name Type Attributes Description
className String | function A string for one or more space-separated classes, or a function that returns one or more space-separated classes, to be toggled from the class attribute.
switch Boolean <optional>
 A Boolean to determine whether the class should be added or removed.
Return Value

The updated form of the original Cheerio matched-set object.

Examples
$body.find(".fruit").toggleClass("pear");
let [b1, b2] = [true, false];
$body.find("div").toggleClass("pear", b1); => adds until b1 becomes false
$body.find("div").toggleClass("blueberry", b2); => removes until b2 becomes true
$body.find(".foo").toggleClass(function() {
  if ($(this).parent().is(".bar")) {
    return "happy";
  } else {
    return "sad";
  }
});
