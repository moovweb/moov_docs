domManipulator(element, position, optionsopt) Source (L600)

domManipulator manipulates the DOM elements moving and creating them

Parameters
Name Type Attributes Description
element Object | String The element tagname or cheerio element object
position String The position to move the element(top, bottom, after, before)
options Object <optional>
 It may be any param, like id, class and there is reserved param called "content" which is the content of the element
Properties
Name Type Attributes Description
content Object <optional>
 The content of the element
Return Value

Returns a cheerio instance

Examples
## Create an element and move it to a position

$('div').domManipulator('span', 'top');
$('div').domManipulator('<span>', 'bottom');
$('div').domManipulator('<span />', 'after');
$('div').domManipulator('<span></span>', 'before');
## Create an element passing some attributes

$('div').domManipulator('span', 'top', { content: 'content of the span' });
$('div').domManipulator('span', 'top',  { content: 'content', class: 'the class', id: 'the id' });
## Move an element to a position

$('div').domManipulator($('span'), 'top');
$('div').domManipulator($('span'), 'bottom');
$('div').domManipulator($('span'), 'after');
$('div').domManipulator($('span'), 'before');

makeToggler(options) Source (L912)

Makes a new toggler It just creates a new Uranium toggler using the default classes and elements of the project, keeping the consistency between them.

Parameters
Name Type Description
options Object Options of the toggler
Properties
Name Type Description
container String This is the container of the toggler, it will be the data-ur-set="toggler"
button String The button of the toggler.
content String The content of the toggler.
skin String The skin of toggler. In your project, you may have some classes like mw-blue-toggler..
keepOpened Boolean Keep all togglers opened by default and doesn't add the Uranium attributes
viewMoreText String It's the text of the main category. It's the main category created by wrapLink
wrapLink String It's for the replace of the link button to a div, it indicates which element the element "a" must be wrapped. That's suppose to remove the mouse click desktop behavior and allow the user click on the main category without redirecting him
Return Value

Returns a cheerio instance

Example
## Simple Toggler

$('ul').makeToggler({
 content: 'li',
 button: 'a',
 content: 'ul'
});
