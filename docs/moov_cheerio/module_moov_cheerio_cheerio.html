dumpTable() → {Cheerio}

Convert the selected table to a div structure. It will convert all table elements to divs and remove the table attributes. Like: table -> div.mw-was-table; thead -> div.mw-was-thead; tbody -> div.mw-was-tbody; tfoot -> div.mw-was-foot; tr -> div.mw-was-tr; td -> div.mw-was-td; th -> div.mw-was-th;
